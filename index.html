Install

If you are using (Surge) for the first time, first open terminal and install
(Surge) by typing this code inside it


surge install code

   
   npm install --global surge 
================================or shortcut code================================
    npm i -g surge

Domain search

After the (surge) installation is complete, open the
project you want to deploy in Terminal and enter this
code


surge domain search code

   
  surge

surge code

Research Domain

surge code

get domain permanent

If you want, you can write a domain for yourself as you like and search to see if it is available. If it is available, it will show success like this image. Then if you want to save that domain forever, then write your code in the terminal.


surge domain save code

   
   
    echo (your domain name ) >CNAME

update your Domain

Many times it is seen that even after deploying the project we have to make many changes in it. To save these changes, we need to update the document later. Using this code below, you will update it in your comments so that your work can be seen on the website.


surge domain update code

   
  surge --domain (your domain name)
